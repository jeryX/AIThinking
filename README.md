# AIThinking

## 知识结构

特定领域的知识结构就是一个小世界。而人类很多工作就是基于这个小世界的改造。

没有清晰的知识结构，自然无法进行清晰的思考。
我们需要从根本上搞清楚事物之间的关联关系，这样AI运行就事半功倍。
这个道理跟编码是一样的。如果我们的思绪就像起伏不平的湖水，我们怎么看清这一切。


## AI是信息处理技术的一次大跨越

在过往很多年，人类大脑一直都是信息处理的核心。
软件让固化的数据处理效率提升了很多倍。但是很多数据仍然需要人来处理。
而AI则进一步放大了计算机的数据处理能力。

人脑所形成的对外部世界的认知是人与人的根本区别，而不同人的认知模型有着较大的区别。
一旦这种处理过程可以用计算机来表达，那么信息处理的速度将会大幅提升。

## 程序与AI

现实中我们会遇到需要严格符合业务逻辑的要求，然后Agent类程序结果并不总是符合预期。
所以到底是可控程序里面包AI，还是Agent使用程序，这是不同的技术选择
